# Villa Capo Grillo Cottage - Next.js per Vercel

Progetto aggiornato con le foto reali della villa e dei paesaggi dell'isola di Vulcano.

## Dove sono le immagini
Le foto sono in:

```txt
public/images
```

Puoi modificarle sostituendo i file con lo stesso nome, oppure cambiando i riferimenti in `app/page.tsx`.

## Avvio locale

```bash
npm install
npm run dev
```

Poi apri:

```txt
http://localhost:3000
```

## Pubblicazione su Vercel

1. Carica il progetto su GitHub
2. Importa il repository su Vercel
3. Deploy automatico

## Form contatti
Il form è grafico: va collegato a un servizio come Formspree, Resend, Brevo o backend personalizzato.
